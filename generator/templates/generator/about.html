About Page

Welcome to MJ's password generator.
This web program enables you to create a random password that generates characters based on your preferences.

How to Use

  1. Go to the homepage.
  2. Enter the number of characters you would want to compose your password.
  3. By default, the generator only provides lower-case letters, but you could check if you would like to create them with upper-case, numbers, or special characters.
  4. Click "Generate Password" to get your random password.
